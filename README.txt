Project Name: TaskForge

Description:
This is a simple team task manager made for my B.Tech assignment. It allows users to create projects, assign tasks and track progress.

Features:
* Login and Signup
* Create Projects
* Add Members
* Task Management
* Dashboard

Tech Used:
Node.js, Express, MongoDB, HTML, CSS, JavaScript

How to Run:
1. npm install
2. npm start
3. open frontend files
